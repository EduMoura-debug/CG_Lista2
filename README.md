Arquivos: dado.py, dado.png

```
python dado.py
```

## 2- Textura – Esfera – “Planeta”

Como executar: No diretório do arquivo, executar o comando: 
```
python mundo.py
```

Arquivos: paraboloide.py

Como executar: No diretório do arquivo, executar o comando: 
```
python paraboloide.py
```
